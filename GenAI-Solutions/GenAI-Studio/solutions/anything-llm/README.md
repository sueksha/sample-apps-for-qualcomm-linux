# AnythingLLM Solution for GenAI Studio
This folder contains the AnythingLLM demo assets for GenAI Studio, including an implementation guide for both the desktop application and Docker, and a patch file for Docker-based deployment.

## What is AnythingLLM?

AnythingLLM is an AI workspace application that helps you:
- chat with different LLM providers (cloud or local)
- create document-aware assistants (RAG)
- organize workspaces, chats, and sources in one desktop app

### Download AnythingLLM for Windows

Use any of the official links below:

- Official Desktop Download Page: https://anythingllm.com/desktop
- Official GitHub Releases Page: https://github.com/Mintplex-Labs/anything-llm/releases
- Official Windows Installation Guide: https://docs.anythingllm.com/installation-desktop/windows

Recommended installation flow:

1. Open the desktop download page or releases page.
2. Download the latest Windows installer (`.exe` or `.msi`) for your system (usually x64).
3. Run the installer and complete the setup wizard.
4. Launch AnythingLLM from Start Menu.
5. On first launch, complete initial setup:
   - select your LLM provider (for example, OpenAI, Azure OpenAI, Ollama, etc.)
   - add the required API key or endpoint
   - create your first workspace

Tip: If your organization blocks unknown installers, ask IT to approve the signed AnythingLLM installer before installation.
